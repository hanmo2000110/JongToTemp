- `/dashboard`
  - 종목 선택 + 핵심 지표 카드(부정비율/3일 변화/평균 게시글/감성지수)

- `/stocks`
  - 검색, 종목 추가, 활성 토글, 순서 변경, 메모 수정, 삭제
- `/`
  - 초기 스캐폴딩 문구 상태(현재 기능 반영 필요)

### 1-2. API/서버 로직
- 종목 API: 목록/생성/수정/삭제/순서변경 구현
- 수집 API: `collect` 트리거 및 종목 필터 처리 구현
- 대시보드 API: `stockCode + range(14d/30d)` 기반 응답 구현
- 서버 유틸: 감성 집계, 가격 이동평균 계산, idempotency 유틸 구현

### 1-3. 데이터/운영 기반
- Supabase 마이그레이션 초안 파일 존재
- 현재 API 저장소는 인메모리 구조(재기동 시 데이터 유실)
